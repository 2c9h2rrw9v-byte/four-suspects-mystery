# 毒の連鎖 ─翌朝の悪夢─

**4人用マーダーミステリー / GMなし / 約2時間**

---

## あらすじ

蒼海町で、同一の手口による連続毒殺事件が発生した。
被害者はいずれも資産家・田中哲夫の関係者。
警察は翌朝には容疑者を逮捕したが、男は「冤罪だ」と叫んだ。
弁護士が動き、男は釈放された。
そして、釈放された翌朝——三人目が死んだ。

真犯人は誰か。この会議室の中にいる。

## 遊び方

このゲームはGMなしで、全員がプレイヤーとして事件の真犯人を追い求めます。

**フェーズ進行：**

| # | フェーズ | 時間目安 | 形式 |
|---|---------|---------|------|
| 1 | オープニング | 10分 | 全員読み合わせ |
| 2 | 配役決め | 5分 | 話し合い |
| 3 | 事件発生 | 10分 | 全員読み合わせ |
| 4 | 個人情報読み込み | 10分 | 各自で読む（秘密） |
| 5 | 全体会議 | 10分 | 自己紹介・基本情報共有 |
| 6 | 調査 | 20分 | 証拠カードを調査 |
| 7 | 第一議論 | 15分 | 議論 |
| 8 | ⭐追加事件 | 5分 | 全員読み合わせ |
| 9 | 追加調査 | 10分 | 新証拠を調査 |
| 10 | 第二議論 | 15分 | 最終議論 |

**合計：約125分（約2時間）**

## プレイのコツ

**読み合わせを大切に** — オープニング・事件発生・追加事件は必ず声に出して読む。担当パートが書いてあります。

**個人情報は秘密** — 自分のキャラクターシートは他の人に見せないこと。

**キャラクターを演じる** — 本音を隠しながら議論に参加しましょう。

**情報を整理する** — メモを取りながら議論すると楽しめます。
