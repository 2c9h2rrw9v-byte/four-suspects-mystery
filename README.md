# 四人の容疑者

## 概要
このマーダーミステリーは、4人で楽しむオフラインゲームです。
GMなしで、全員がプレイヤーとして事件の真犯人を追い求めます。

**プレイ時間：** 約2時間
**プレイ人数：** 4人推奨
**難易度：** 中～高

---

## 遊び方

このゲームは以下のフェーズで進行します：

1. **オープニング** → 全員で読み合わせ
2. **配役決め** → 各自の役職を確認
3. **事件発生** → 全員で読み合わせ
4. **個人情報読み込み** → 自分の情報のみ確認
5. **全体会議** → 質問・回答
6. **調査** → 証拠確認
7. **第一議論** → 犯人は誰か議論
8. **追加事件** → 新しい事件が発生！全員で読み合わせ
9. **追加個人情報読み込み** → 新しい秘密を確認
10. **追加調査** → さらに情報収集
11. **第二議論** → 本当の犯人は誰か再議論
12. **犯人投票** → 投票で決定
13. **エンディング** → 真実の暴露

## プレイのコツ

✅ **読み合わせはセリフで楽しく** - オープニングと事件発生、追加事件は全員で声に出して読むと盛り上がります

✅ **個人情報は秘密** - 自分の役職情報は他の人に見せないこと

✅ **質問は自由** - 調査フェーズでは、好きなだけ質問できます

✅ **議論を楽しむ** - 犯人投票までは、議論を最大限楽しみましょう
